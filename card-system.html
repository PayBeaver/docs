发卡系统对接教程

风铃发卡系统

项目地址：Tai7sy/card-system

风铃系统最新的 Gateway 已经支持 PayBeaver 海狸支付，PayBeaver/Api.php，若你的 Gateway 比较老请手动更新。

PayBeaver 海狸自有支付协议

进入后台 -> 渠道设置 -> 添加

名称： PayBeaver 驱动： PayBeaver 方式： PayBeaver 配置： JSON

{
  "app_id": "App ID",
  "app_secret": "App secret"
}

易支付支付协议

进入后台 -> 渠道设置 -> 添加

驱动： Epay 方式： alipay 配置： JSON

{
  "gateway": "https://api.paybeaver.com/api/v1/invoices",
  "pid": "App ID",
  "key": "App secret"
}
