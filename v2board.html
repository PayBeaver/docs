V2Board

PayBeaver for V2Board

PayBeaver 支持多种对接的协议，你可以根据自己的实际情况进行选择。

mGate 支付协议

mGate是一套支付系统，可能有多个支付商使用该系统，使用该系统的均可以使用本接口。

进入 V2Board 后台 -> 支付配置 -> mGate

URL： https://api.paybeaver.com

AppID： 商户中心 -> 开发者 -> App ID

AppSecret：商户中心 -> 开发者 -> App Secret

易支付支付协议

易支付是一套支付系统，可能有多个支付商使用该系统，使用该系统的均可以使用本接口。

进入 V2Board 后台 -> 支付配置 -> 易支付

URL： https://api.paybeaver.com/api/v1/invoices

PID： 商户中心 -> 开发者 -> App ID

Key：商户中心 -> 开发者 -> App Secret
