PayBeaver for SSPanel

1. 下载对应版本的 Pay Beaver SDK

  • SSPanel UIM 版
  • SSPanel Malio 版
  • SSPanel Cool 版
  • SSPanel Cur 版
  • SSPanel Metron 版 (点击查看 SDK 和对接文档)

2. 在项目目录下解压并覆盖，例如 /www/wwwroot/xxx

3. 打开 config/.config.php
找到 $_ENV['payment_system']
或者 $System_Config['payment_system']
将值修改为 paybeaver

4. 并在 config/.config.php 下方插入以下内容

注：若你的变量名都是 $System_Config 的话，则将下面三行的 $_ENV 替换为 $System_Config

$_ENV['paybeaver_app_id'] = 'App ID';
$_ENV['paybeaver_app_secret'] = 'App Secret';
$_ENV['paybeaver_url'] = 'https://api.paybeaver.com';

PayBeaver for Metron theme

1. 下载 Pay Beaver SDK

2. 在项目目录下解压并覆盖，例如 /www/wwwroot/xxx

3. 打开 config/.config.php

找到 PcexPay 并填上以下内容：

$_ENV['pcexpay_id']   = 'App ID';
$_ENV['pcexpay_key']  = 'App Secret';

4. 打开 config/.metron_setting.php

$_MT['pay_wxpay']   = 'paytaro';
$_MT['pay_alipay']  = 'paytaro';
